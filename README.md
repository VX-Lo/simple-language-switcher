# Simple Language Switcher

**Simple Language Switcher (SLS)** is a lightweight World of Warcraft addon that makes it easy to change your character’s **spoken language** using simple **slash commands** or a **compact minimap button**.


---


## Features

- **Fast language switching** via `/sls ` (e.g. `/sls common`, `/sls darnassian`)  
- **Cycle through known languages** using `/sls cycle`  
- **List all known languages** with `/sls list`  
- **Optional minimap button** for one-click switching and selection menu  
- **Persists your choice** automatically between sessions  
- **Fuzzy input matching** (no need to type full names)


---


##  Slash Commands

| Command        |Description                                           |
| -------------- |----------------------------------------------------- |
| `/sls`         |Shows help and available commands                     |
| `/sls `        |Switch to a specific language (partial names allowed) |
| `/sls cycle`   |Cycle to next known language                          |
| `/sls list`    |List all known languages                              |
| `/sls minimap` |Toggle the minimap icon                               |
